SETUP

Posted on Jan 1, 0001
HUGO Go-lang-1.11 is necessary for Hugo. In Ubuntu-18.04 we need to install it from the repository below sudo add-apt-repository ppa:longsleep/golang-backports sudo apt-get update sudo apt-get install golang-go After that we can install the latest hugo binary (v0.53) wget https://github.com/gohugoio/hugo/releases/download/v0.53/hugo_0.53_Linux-64bit.deb sudo dpkg -i hugo_0.53_Linux-64bit.deb Hugo themes can be downloaded directlry into the working directory git clone --depth 1 --recursive https://github.com/gohugoio/hugoThemes.git themes (message "Hello")
